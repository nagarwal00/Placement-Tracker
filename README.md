# 🎓 Placement Tracker

A modern Flask-based web application to help students manage campus placements efficiently.

## 🚀 Features

- 🔐 User Login & Registration
- 🏢 Company Management
- ✏️ Edit Company Details
- 📊 Dashboard with Statistics
- 📈 Preparation Progress Tracking
- 🔍 Search Companies
- 📅 Upcoming Drives
- 👥 Multi-user Support

## 🛠️ Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap 5
- Jinja2
